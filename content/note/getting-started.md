This is my first note. Hugo with the [hugo-ivy](https://github.com/yihui/hugo-ivy) theme looks clean.

## How to create a new note

```bash
hugo new content note/my-note-title.md
```

Then edit the file and add `categories` and `tags` in the frontmatter.

## How to preview

```bash
hugo server -D
```
